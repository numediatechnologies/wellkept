# Well-Kept

Well-Kept is a South Africa-focused recommerce platform for appliances and furniture.

## Stack

- React + Vite + TypeScript
- Vercel Functions
- Supabase Postgres/Auth/Storage
- Resend email

## Key Features

- Guided buyer wanted-post wizard
- Guided seller listing wizard
- Brand voice enforcement for generated copy
- AI-style suggestion endpoints with deterministic fallback heuristics
- Geography-aware sitemap pages for South Africa
- Seller alerts and notification center
- Payfast + EFT-ready payment endpoints

## Development

```bash
npm install
npm run dev
```

## Environment

Copy `.env.example` to `.env.local` and fill in the required values.

## Supabase

Schema and seed files live under `supabase/`.
